# 🏭 门卫操作卡

> 贴在门岗电脑旁 · 每步照做即可

---

## ▶ 访客入场（每位访客都做）

1. 电脑浏览器打开管理页面，登录
2. 选择访客要去的 **区域**
3. 点击 **生成入场码**，屏幕出现二维码
4. 让访客用手机 App **扫这个码**
5. 访客手机显示「正在为您办理入场」→ **等待**
6. 手机自动跳转设置页面 → **不要碰访客手机，等它自己完成**
7. 手机回到 App 并显示管控界面 → **入场完成** ✅
8. 电脑上该访客状态变成 `restricted` → 确认成功

⏱ 正常等待 1–2 分钟。**超过 3 分钟没反应 → 看下面「异常处理」**

---

## ◀ 访客离场（每位访客离开时都做）

1. 在电脑管理页面找到该访客
2. 点击该访客的 **离场码**，屏幕出现二维码
3. 让访客用手机 App **扫这个码**
4. 手机显示「正在办理离场」→ 等几秒
5. 手机显示「离场已办理，感谢配合」→ **离场完成** ✅
6. 让访客试一下相机能不能拍照 → 能拍 = 正常

---

## 🔧 首次设备授权（只做一次）

新访客第一次用 App 时，手机会提示开启「无障碍服务」：

1. 按手机屏幕提示，进入 **设置 → 无障碍**
2. 找到 **Factory Control**，点击 **开启**
3. 回到 App，后续全部自动完成

> 这个步骤只需要做一次，以后再来不用重复。

---

## ⚠ 异常处理

| 手机上看到什么 | 怎么办 |
|--------------|--------|
| 一直转圈超过 3 分钟 | **长按**屏幕中间文字 5 秒 → 出现更多按钮 → 点「重试连接」 |
| 「设置已中断，如需帮助请联系门卫」 | 去手机设置 → 无障碍 → 重新开启 Factory Control |
| 「当前正在使用中」 | 正常！说明上次没离场。先在电脑上点「强制离厂」，等手机回到首页再重新扫码 |
| 离场后相机还是不能用 | 让访客 **重启手机**，重启后自动恢复 |
| 电脑页面打不开 | 刷新页面；还不行就联系工程师 |

---

## ❌ 常见错误提醒

- **不要**在自动化过程中触碰访客手机
- **不要**替访客关闭 App 或切换到其他应用
- **不要**让访客在入场完成前断开 WiFi
- 访客手机必须连接 **工厂 WiFi**，不能用自己的流量
